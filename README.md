✨ Features
🔐 Secret Code Unlock – Only someone with the code sees the full experience

💌 Love Letter & Sweet Notes – Add your own heartfelt words

📸 Memories Gallery – Showcase your favorite photos together

🎵 "Our Song" Area – Embed a special track that plays right on the page

🌸 Soft, Romantic Design – Gentle colors and elegant layout

⚡ Pure HTML/CSS/JS – No frameworks, no dependencies, no complexity

🛠️ How to Customize
Clone or download the repository

Open index.html in any text editor

Replace placeholder content:

Secret code – change the password in the JavaScript section

Photos – swap out photo1.jpg through photo4.jpg with your own images

Your song – replace your-song.mp3 with your chosen track

Love letter & notes – edit the text to say what's in your heart

Save and open index.html in a browser

🎁 Share the Love
Once customized, send the file, host it online, or surprise your special someone with a link. It's meant to be simple, personal, and memorable.
